A minimal agent stack. Two engineering decisions worth noting.

Loading architecture...

Serverless + SSE

Aliyun FC terminates streaming responses early. Solved by collecting all SSE chunks server-side before flushing — the client still receives typed events and renders progressively. Same challenge I solved for a different reason at Zalando.

Multi-provider fallback

Primary LLM is Qwen-Turbo via OpenAI-compatible API. Automatic fallback to DeepSeek then OpenAI — all providers share the same async httpx client and Pydantic tool schemas. Swapping is config-only, no code changes.
